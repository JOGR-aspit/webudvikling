# W1: Introduction to Web Development

## Course Overview

Welcome to W1 - Introduction to Web Development! This course focuses on the fundamentals of building web pages using **HTML** and **CSS**. You'll learn how to create beautiful, structured web pages from scratch.

## What You'll Learn

By the end of this course, you will be able to:

- [ ] Create HTML documents with proper structure
- [ ] Use HTML tags to organize content (headings, paragraphs, lists, links, images)
- [ ] Apply CSS styling to control appearance (colors, fonts, spacing, layout)
- [ ] Build complete web pages that look professional
- [ ] Debug and fix common HTML and CSS issues

## Prerequisites

**None!** This course is designed for beginners with no prior web development experience.

## Course Structure

### Curriculum

The `curriculum/` folder contains all learning materials:

1. **HTML Basics** - Understanding HTML structure and common tags
2. **HTML Elements** - Working with headings, paragraphs, lists, and links
3. **Images and Media** - Adding images and other media to your pages
4. **CSS Fundamentals** - Introduction to CSS and how to apply styles
5. **CSS Typography** - Styling text with fonts, colors, and spacing
6. **CSS Layout** - Using margins, padding, borders, and positioning
7. **CSS Flexbox** - Creating flexible layouts
8. **Responsive Design** - Making pages work on different screen sizes

### Assignments

The `assignments/` folder contains practice exercises to reinforce your learning:

- **Assignment 1**: Your First Web Page
- **Assignment 2**: HTML Elements Practice
- **Assignment 3**: Styling with CSS
- **Assignment 4**: Building a Complete Page
- **Assignment 5**: Responsive Layout Project

*Solutions are available in `assignments/solutions/` for reference.*

### Projects

The `projects/` folder contains larger, practical projects:

- **Project 1**: Personal Profile Page
- **Project 2**: Product Landing Page
- **Project 3**: Simple Portfolio Website

### Code Examples

The `code-examples/` folder contains extensive, well-commented code examples for every topic covered in the course.

## Course Policies

### Working with Code

- **Always test your code** - Open your HTML files in a browser to see the results
- **Use browser developer tools** - Press F12 to inspect elements and debug
- **Save your work frequently** - Don't lose your progress
- **Comment your code** - Explain what each section does (for your future self!)

### Getting Help

If you're stuck:

1. **Re-read the curriculum** - The answer might be there
2. **Check the code examples** - Similar examples can help you understand
3. **Try the troubleshooting section** - Common issues and their fixes
4. **Compare with solutions** - See how the problem was solved
5. **Ask for help** - Don't hesitate to reach out

## Tips for Success

### Learning HTML

- **Practice frequently** - Write HTML every day if possible
- **Use semantic tags** - Choose the right tag for the job
- **Validate your code** - Use online validators to check for errors
- **Learn by copying** - Start by copying examples, then modify them

### Learning CSS

- **Experiment freely** - Try different values to see what happens
- **Use browser tools** - Inspect elements to understand styles
- **Start simple** - Master basics before moving to advanced topics
- **Practice layout** - Spend time on positioning and flexbox

### General Advice

- **Be patient** - Web development takes time to master
- **Break problems down** - Divide complex tasks into smaller steps
- **Celebrate small wins** - Every working line of code is progress
- **Ask questions** - Curiosity leads to deeper understanding
