# Example Solution Images

This folder should contain images for the Paris travel guide example.

## Required Images

For the example solution to display correctly, you need the following images:

1. **paris-skyline.jpg** - A photo of the Paris skyline featuring the Eiffel Tower
2. **eiffel-tower.jpg** - A photo of the Eiffel Tower, preferably at night

## Where to Find Images

You can use:

- Your own travel photos
- Free stock photo websites like:
  - [Unsplash](https://unsplash.com/)
  - [Pexels](https://www.pexels.com/)
  - [Pixabay](https://pixabay.com/)

Search for "Paris skyline" and "Eiffel Tower night" to find suitable images.

## Image Guidelines

- Images should be appropriately sized (800-1200px wide is good for web)
- Use descriptive file names in kebab-case
- The alt text in the HTML already describes what these images should show

## Note

Images are not included in this repository due to file size and licensing considerations. Instructors should add appropriate images before using this example solution with students.
