Welcome to My Website

This is a complete page layout demonstrating how block and inline-block elements work together. The navigation bar above uses display: inline-block to create horizontal menu items that accept padding.

Block elements like this container stack vertically and take full width. They're perfect for creating page sections like the header, navigation, main content, and footer.

What You'll Learn

By completing this exercise, you'll understand how to:

  • Create horizontal navigation using inline-block
  • Style links with padding for larger click areas
  • Build a complete page layout with multiple sections
  • Center content using auto margins

The Power of Inline-Block

Inline-block combines the best of both worlds: elements stay on the same line like inline elements, but they accept width, height, and padding like block elements.

This makes inline-block perfect for navigation menus, button groups, and any situation where you need elements side by side with control over their sizing.
