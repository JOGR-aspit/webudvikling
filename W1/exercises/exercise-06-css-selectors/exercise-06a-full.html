Class Selector Example

This card is styled with a class selector (.card { }). Any element with class="card" gets this style.

Multiple elements can share the same class. This makes styles reusable!

The header above uses an ID selector (#main-header { }). IDs should be unique - only one element per page has this ID.

Selector Chaining Example

Look at these buttons. They all have class="button":

Regular Button Large Button Large Primary Button

The buttons use selector chaining:

  • .button - base button style
  • .button.large - large button (chained)
  • .button.primary - primary button (chained)
  • .button.large.primary - large AND primary (double chained)

Descendant Combinator Example

These paragraphs are styled with a descendant combinator (.card p { }).

This means: style paragraphs ONLY when they're inside .card elements. Paragraphs outside .card get the default p style.

This paragraph has an additional class="highlight". It gets both .card p styles AND .highlight styles.

This link is styled with .card a { } - descendant combinator.

Summary of Selector Types

Here's what we've demonstrated:

  • Element selector: p { } - styles all paragraphs
  • Class selector: .card { } - styles elements with class="card"
  • ID selector: #main-header { } - styles unique header
  • Selector chaining: .button.large { } - styles elements with both classes
  • Descendant combinator: .card p { } - styles paragraphs inside cards
